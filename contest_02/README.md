# 🅰️ A — Минимум на стеке

## Условие
Реализовать структуру данных, поддерживающую:
1. Добавление элемента  
2. Удаление последнего элемента  
3. Получение минимального элемента  

## Входные данные
- `n` — число операций (`1 ≤ n ≤ 10^6`)
- Далее `n` строк:
  - `1 x` — добавить элемент  
  - `2` — удалить элемент  
  - `3` — вывести минимум  

## Выходные данные
Для каждой операции `3` вывести минимум.

---

# 🅱️ B — Минимум на отрезке

## Условие
Дан массив длины `N`. Рассматривается окно длины `K`, которое двигается вправо.  
Нужно найти минимум в каждом положении окна.

## Входные данные
- `N, K` (`1 ≤ K ≤ N ≤ 150000`)
- массив из `N` чисел

## Выходные данные
`N - K + 1` чисел — минимумы для каждого окна.

---

# 🅲 C — Постфиксная запись

## Условие
Вычислить значение выражения, записанного в постфиксной форме.

## Входные данные
- строка с числами и операциями `+ - *`

## Выходные данные
- значение выражения

# 🅳 D — Шарики

## Условие
Дана последовательность шариков.  
Если есть подряд ≥3 шариков одного цвета — они удаляются.

Нужно определить, сколько шариков будет уничтожено.

## Входные данные
- `n` — количество шариков (`n ≤ 100000`)
- цвета шариков

## Выходные данные
- количество уничтоженных шариков

---

# 🅴 E — Сортировка вагонов

## Условие
Дан порядок вагонов.  
Можно использовать тупик (стек), чтобы переставить их в порядок `1..N`.

## Входные данные
- `N` (`1 ≤ N ≤ 2000`)
- перестановка чисел `1..N`

## Выходные данные
- последовательность операций  
или  
- `0`, если невозможно

---

# 🅵 F — Астроград

## Условие
Нужно обработать очередь с операциями:
1. добавить в конец  
2. удалить с начала  
3. удалить с конца  
4. узнать позицию элемента  
5. узнать первый элемент  

## Входные данные
- `n` (`1 ≤ n ≤ 10^6`)
- описание операций

## Выходные данные
Ответы для операций `4` и `5`

---

# 🅶 G — Гоблины и очереди

## Условие
Очередь с тремя операциями:
- `+ x` — в конец  
- `* x` — в середину  
- `-` — удалить из начала  

## Входные данные
- `n` (`1 ≤ n ≤ 100000`)
- операции

## Выходные данные
Для каждой операции `-` вывести удалённый элемент

# 🅷 H — Хорошие дни

## Условие
Найти подотрезок массива, где значение: сумма * минимум максимально.

## Входные данные
- `n` (`1 ≤ n ≤ 100000`)
- массив

## Выходные данные
