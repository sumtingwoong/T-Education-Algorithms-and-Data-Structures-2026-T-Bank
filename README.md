# Algorithms and Data Structures — T-Bank Education 2026

Репозиторий с решениями задач курса  
**Алгоритмы и структуры данных (АИСД)** от T-Bank Education, 2026.

## Структура

Репозиторий организован по контестам:

- каждый контест находится в отдельной папке `contest_XX`
- внутри контеста:
  - `A.py`, `B.py`, ... — решения задач
  - `README.md` — условия, примеры и комментарии

Количество контестов добавляется по мере прохождения курса.

## Язык
- Python 3

## Цель
- практика алгоритмов и структур данных
- подготовка к контестам и экзаменам
- аккуратная систематизация решений

> Учебный репозиторий. Решения предназначены для личного использования.
